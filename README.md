# To-Do-ListApp

## Description
A simple web application built using HTML and JavaScript that allows users to **add, view, and manage daily tasks**. It demonstrates basic DOM manipulation and interactive web functionality.

## Features
- Add new tasks  
- View tasks in a list  
- Simple, clean, and interactive interface  

## Technologies Used
- HTML  
- JavaScript  

## How to Use
1. Open `index.html` in your browser.  
2. Type a task in the input box.  
3. Click **Add Task** to display it in the list.  
4. Repeat to add more tasks.  

## Example Output
- User types: `Complete Assignment` → Task appears in the list.  
- User types: `Practice Python` → Task appears below the first task.
